# teamfinder
  
   
   1. clone repository
   2. Install the required packages
   ```
   cd client  
   npm install --legacy-peer-deps
   cd backend
   npm install
   ```
   3. run backend at /backend
   ```
   npm run start:dev
   ```
   Backend will run at localhost:3002.
  
  4. run frontend at /client
  ``` 
  npm start
  ```
  Frontend will run at localhost:3000.
